# VoidRP WebGUI (NeoForge)

NeoForge 1.21.1 server-side mod that exposes an in-game web interface for administrative and player-facing features.

**Stack:** NeoForge 1.21.1 · Java 21 · Gradle

## Build

```bash
./gradlew build
# Output: build/libs/webgui-neoforge-*.jar
```

## Deploy

Copy the built JAR to your NeoForge server's `mods/` directory.

## Requirements

- NeoForge 21.1.x
- Minecraft 1.21.1
